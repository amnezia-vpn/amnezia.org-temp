Guide. Install ShadowSocks with the Cloak plugin for Android

EnglishEn
  • Русский Ru
The Amnezia client does not yet support installing the ShadowSocks protocol with the Cloak plugin in the application itself, However, such a bundle can be created using the Amnezia client and the original ShadowSocks client.

1. Buy a VPS from any hosting provider (Amnezia works with any Ubuntu server (officially supported version is 20.04), also supported by Debian 10).

2. Download Amnezia application on Windows, connect to the server through it. Enter VPS data into the application, select "Configure VPN protocols manually", choose OpenVPN over Cloak protocol.

3. Install the official ShadowSocks client from "Max Lv" from Google Play on Android

4. Install Clock plugin on Android from here, APK file. You will have to allow installation of APKs from unknown sources.

5. Open Amnezia on Windows - Select OpenVpn over Clock protocol - Share connection for Shadowsocks

6. Open ShadowSocks client on mobile - Add profile - Scan QR code from Windows client, scanning

7. Open Amnezia on Windows - Select OpenVpn over Clock protocol - Share connection for Cloak

8. Open ShadowSocks client on mobile - Go to the created profile - Select Cloak plugin - Configure

Copy and paste the parameters there: UID, Public Key and Server Name from Amnezia on Windows
