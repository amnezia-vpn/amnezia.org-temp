Guide. Install ShadowSocks with the Cloak plugin for Windows

EnglishEn
  • Русский Ru
The Amnezia client does not yet support installing the ShadowSocks protocol with the Cloak plugin in the application itself, However, such a bundle can be created using the Amnezia client and the original ShadowSocks client.

1. Buy a VPS from any hosting provider (Amnezia works with any Ubuntu server (officially supported version is 20.04), also supported by Debian 10 and Centos 8).

2. Download Amnezia application on Windows, connect to the server through it. Enter VPS data into the application, select "Configure VPN protocols manually", choose OpenVPN over Cloak protocol.

3. Download and unzip the ShadowSocks client, we need the .exe file.

4. Download Cloak for Windows (ck-client-windows-386 for 32-bit systems or ck-client-windows-amd64 for 64-bit systems)

5. Move the installed Cloak plugin (i.e. just an .exe file) to the Shadowsocks folder unpacked earlier - the plugin will be in the same folder as Shadowsocks.exe.

6. Rename the plugin to "cloak" (without quotes, just cloak)

7. Launch Shadowsocks on Windows, right-click on it in the tray and click on "Associated ss:// links"

8. Open in "Windows" Amnezia - "Share connection" - "Share connection for Shadowsocks" . First select OpenVPN over Cloak protocol, then go to settings, select “Share connection” => “Share for ShadowSocks”

9. Click "Copy string", paste it into the browser address bar, agree to open the link in the application. Shadowsocks server editor will open - there will be pop-up windows, click ok everywhere "ок"

10. The "IP address, port, password and encryption" fields should be filled in automatically. Change the "Port" and "Proxy port" to 443, write cloak in the "Plugin Program" field

11. Open Amnezia in Windows, "Settings" => "Share connection" => "Share connection for Cloak", click Copy.

12. below in notepad insert the following line (without spaces and hyphenation)

UID=*;AlternativeNames=;PublicKey=*;KeepAlive=0;ServerName=*;EncryptionMethod=aes-gcm;BrowserSig=chrome;Transport=direct;CDNOriginHost=;ProxyMethod=shadowsocks;NumConn=4

In the line for UID, PublicKey and ServerName (instead of *) we paste the values of UID, PublicKey and ServerName from the text copied from Amnezia (what comes after the colon is in quotes, but we copy without quotes - only the value itself).

13. Select the line from item 12 (with inserted values of UID ServerName and PublicKey), copy it

14. Go back to the Shadowsocks server editor and in the "Plugin options" line paste the copied string

15. Check that the string is inserted correctly - it should start with UID and end with NumConn=4.

16. Click apply, ok. If there is a server with the name "New server" in the list of servers - it can be deleted, one server will remain in the list - its IP will be displayed

17. In the tray, click on the Shadowsocks app - "System Proxy" - "Global"
